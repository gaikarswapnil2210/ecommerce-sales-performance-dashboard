E-Commerce Sales Performance Dashboard (2023–2025)

-Project Overview :
This project focuses on analyzing e-commerce sales data to understand overall business performance,
product category contribution, geographic sales distribution, and monthly sales trends.
The goal is to provide an executive-level dashboard that supports data-driven decision-making.

-Tools & Technologies :
SQL Server – Data storage and analysis | Power BI – Data modeling, DAX, and dashboard development | Power Query – Data cleaning and transformation

-Dataset Description :
The dataset contains 50,000+ e-commerce transactions with the following key attributes:
Transaction ID,Customer details (age, country),Product category,Sales amount,Payment method,Transaction date

-Data Cleaning & Preparation :
Removed null and invalid sales values,Handled age outliers to maintain realistic customer demographics,Standardized column names and data types,Created a Date Table for accurate time-based analysis

-Key KPIs:
1)Total Sales: $25M+,2)Total Transactions: 50K+,3)Average Order Value (AOV): $500+.

-Dashboard Features :
KPI cards for quick business overview,Sales by Country analysis,Sales by Product Category performance,Monthly Sales Trend with year-wise comparison,Interactive slicers for Year, Country and Product Category.

-Key Insights:
Sales are distributed across multiple product categories with consistent contribution,
Certain countries contribute significantly higher revenue,
Monthly trends highlight seasonal patterns and stable performance over time.

-Business Value :
This dashboard enables stakeholders to : Monitor overall sales performance,Identify top-performing markets and product categories,Analyze trends over time for strategic planning.
